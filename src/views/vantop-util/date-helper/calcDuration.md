## calcDuration 方法
::: tip 常规用法
calcDuration(ms): string

返回一个表示时长的字符串(hour'min''sec)。
:::

### calcDuration 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| ms         | 毫秒     |  number      | -      |


#### 例子

```js
calcDuration(7482220)
// --> 2'4''42

```
