## ImgPreviewer 图片预览

### 基础用法
:::demo

```html
<template>
    <ImgPreviewer url="http://www.leadnov.com/Upload/20190428165739.gif"/>
```

### ImgPreviewer Attributes

| 参数名称 | 说明               |类型               | 默认值 |

| url      | 图片地址 | String        |null      |
